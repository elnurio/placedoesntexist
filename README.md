# Studio — сайт архитектурной студии

Статический сайт (HTML/CSS/JS, без сборки).

## Структура

```
index.html        — вся разметка, одностраничный сайт с секциями
css/style.css      — стили (переменные/токены дизайна вверху файла)
js/main.js         — мобильное меню, шапка при скролле, reveal-анимации
assets/images/     — сюда добавлять реальные фото/логотипы партнёров
```

## Запуск локально

Любой статический сервер, например:

```
npx serve .
```

или просто открыть `index.html` в браузере.

## Фото

Hero и секция «Италия» ищут файлы:

```
assets/images/hero.jpg   — полноэкранное фото на первом экране
assets/images/italy.jpg  — фото в секции «Итальянская архитектура»
```

Пока файлов нет — показывается декоративный градиент-заглушка (CSS
background с несколькими слоями, картинка и градиент в одном свойстве
`background`, так что при отсутствии файла страница не ломается).
Добавьте файлы с этими именами в `assets/images/` — подхватятся
автоматически, менять код не нужно.
