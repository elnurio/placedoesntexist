# Place Doesn't Exist — сайт архитектурной студии

Статический сайт (HTML/CSS/JS, без сборки).

## Структура

```
index.html      — вся разметка, одностраничный сайт с секциями
css/style.css   — стили (переменные/токены дизайна вверху файла)
js/main.js      — меню, шапка при скролле, hero-слайдшоу, reveal-анимации
content/        — фото
```

## Запуск локально

Любой статический сервер, например:

```
npx serve .
```

или просто открыть `index.html` в браузере.

## Фото

```
content/maindesktop.webp  — слайд hero
content/mainmob.webp      — слайд hero
content/italy.jpg         — фото в секции «Итальянская архитектура» (пока не добавлено,
                             показывается градиент-заглушка, страница не ломается без файла)
```

Оба hero-файла крутятся как слайды (кроссфейд каждые 5 секунд, см.
`js/main.js`). Чтобы добавить ещё слайд — добавьте новый `<img class="hero-slide">`
в `.hero-slideshow` в `index.html`, JS подхватит его автоматически.
