# Studio — сайт архитектурной студии

Статический сайт (HTML/CSS/JS, без сборки).

## Структура

```
index.html        — вся разметка, одностраничный сайт с секциями
css/style.css      — стили (переменные/токены дизайна вверху файла)
js/main.js         — мобильное меню, шапка при скролле, reveal-анимации
assets/images/     — сюда добавлять реальные фото/логотипы партнёров
```

## Запуск локально

Любой статический сервер, например:

```
npx serve .
```

или просто открыть `index.html` в браузере.

## Фото

Все изображения лежат в `content/`:

```
content/maindesktop.webp  — hero, десктоп/планшет (> 720px)
content/mainmob.webp      — hero, мобильные (≤ 720px), переключается через <picture>
content/italy.jpg         — фото в секции «Итальянская архитектура» (пока не добавлено,
                             показывается градиент-заглушка, страница не ломается без файла)
```

Чтобы добавить/заменить фото — положите файл с нужным именем в
`content/`, код менять не нужно.
